# React Vite Tailwind Template

A minimal starter template combining React, TypeScript, Vite, and Tailwind CSS. Fast HMR development with sensible defaults so you can focus on building.

## Getting started
Prerequisites: Node.js (LTS recommended) and npm.

1. Clone the repository:

```bash
git clone <repo-url>
cd React-Vite-Tailwind-Template
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

Open http://localhost:5173 to view the app. Edit `src/App.tsx` or other files—hot reload is enabled.

## Available scripts
- `npm run dev` — Start Vite dev server
- `npm run build` — Build production bundle
- `npm run preview` — Preview production build locally
- `npm run lint` — Run ESLint

## Project structure
- `index.html` — Vite entry
- `src/main.tsx` — App bootstrap
- `src/App.tsx` — Example root component
- `src/index.css`, `src/App.css` — Global and app styles (Tailwind entry)
- `vite.config.ts` — Vite configuration
- `eslint.config.js` — ESLint configuration
- `tsconfig.*.json` — TypeScript configs
